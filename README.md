# restAPY
**Version 1.0.0**

A python module for building Rest APIs

## Installation

```python
import restAPY
```



## Dependencies

#### Python 3.6 (or higher) with the following modules:

- socket
- threading
- json

All of these are part of the default python install and therefore don't need to be installed separately.
